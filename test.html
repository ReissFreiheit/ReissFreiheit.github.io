Advice from Friends

Keep in mind all HTML tags (tables [and all tags in them], headers, divs, spans, etc.)

You can loop through elementsByID - Useful for FOR loops

If swapping out text nodes, you should remove the current one before adding the new one

If looping through inputs, make sure you are targeting the right group of inputs

FOR loops are your best friend

If not HTML attribute, try addEventListener (attachEvent if IE)

Columns are just elements in every row
