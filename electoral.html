Electoral

Overview

Electoral is designed to be a mobile app that enables users to keep up to date on elections around the world. Users can favorite countries and be updated on what elections upcoming. This was a class project from Spring of 2018.

The Problem

Friends of mine who are interested in geopolitics have told me that they have a hard time keeping up with elections worldwide. It is difficult to be up to date on the news of many different countries and their elections. Electoral is meant to enable users to keep track of elections without the hassle of scanning through many different news sources for each different country.

The Process

First I talked with my friends who were interested in the idea in order to get a set of basic requirements. I ended up with the goal statement:

"Electoral will list all upcoming elections in all relevant countries. Countries will be able to be filtered by regions (EU, North America, East Asia, etc.) and users will be able to create their own filters. Electoral will provide users with as much relevant information as possible about an election: which parties are running, party stances, polling data, etc."

I'm a visual thinker, so I needed to sketch out some basic ideas of how this might work.

These vague ideas needed to be refined and tested, so I made an interactive lofi prototype in InVision and asked some classmates and friends to help test the design. I had two participants.

I explained the purpose of the app to the users and had them explore. They each completed the SUS (System Usability Survey) and gave me feedback through comments. Their SUS scores suggested that the system was pretty easy to learn and use (80/100 and 85/100).

I was pretty excited to see the results of the user tests. I had made something people could understand through context rather than a boring tutorial.

The Result

The final part of this project was a HiFi prototype (above) and more user testing. A full breakdown of the project can be found here.

I learned quite a bit about user testing from this project. The most important takeaway was better insight in how to take user feedback and use that to improve on a design.
