Fluent Design File Explorer

The Windows File Explorer is one of the most widely used interfaces in the world. When Microsoft initially announced the Fluent Design System, I was excited to see how they would implement it across the Windows 10 OS. This redesign is just one way I imagine the File Explorer could be transformed.

I wanted to focus on the features of the Fluent Design System that were easily marketable. "Reveal" provides more discoverability to the user. By highlighting the edges of button containers, the user finds out that something is interactable without the UI being cluttered with lines and borders. "Acrylic" is the blurred material effect on the sidebar. It makes the program appear to have its own space on the desktop.
