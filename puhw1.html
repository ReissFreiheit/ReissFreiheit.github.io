Goals

While the information era has made it possible for the average person to be informed on elections, the information is buried. Keeping track of elections across the world is a tedious process. Electoral is an app that aims to change that.

Approach

Electoral will list all upcoming elections in all relevant countries. Countries will be able to be filtered by regions (EU, North America, East Asia, etc.) and users will be able to create their own filters. Electoral will provide users with as much relevant information as possible about an election: which parties are running, party stances, polling data, etc.

Audience

This app is intended to provide information to anyone with an interest in elections, government, and geopolitics. Information will be presented in an accessible manner. There may be a learning curve with terms used based on the context, so Electoral will provide a dictionary of what terms mean in context.

Scope

Electoral aims to be unbiased in its presentation. Electoral will only provide data about an election. Electoral aims to represent all parties in an objective manner. Electoral will not focus on drama in media, only stated positions. Electoral does not aim to influence elections, only provide data. Electoral aims to be a tool used by observers, not necessarily participants.

About
